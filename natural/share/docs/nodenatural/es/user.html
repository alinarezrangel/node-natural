Datos de usuario

En el panel de la izquierda (generalmente en azul) puedes clickear el icono "Usuario" para abrir un nuevo panel. En este panel tu puedes ver tu información de usuario como nombre, nombre de usuario o el Token de Seguridad de Natural. Tu no deberías nunca compartir o utilizar en otros sitios este token. Aquí, en la parte inferior puedes clickear tres botones: el boton de configuración abrirá la configuración del escritorio, el boton de ayuda abrirá esta ayuda y el boton de cerrar sesión mostrará un dialogo de cerrar sesión donde puedes seleccionar entre permanecer en Natural o Salir.

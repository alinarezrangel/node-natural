Archivos

Tu puedes abrir la aplicación NFiles para ver tus archivos en una forma de navegación. Esta es una aplicación de una sola ventana.

Cuando tu clickeas un archivo, el dialogo de NFileOpen se podrá ver, clickea la aplicación que deseas utilizar para abrir el archivo y clickea "Abrir". Para archivos de texto, puedes utilizar NEdit y para imagenes NPixel.
